# Аудит UI многодорожечного плеера ClickForge

## 1. Цветовая палитра и opacity

### Текущая шкала opacity (Color.primary):
| Значение | Использование |
|----------|---------------|
| 0.04 | Заголовок таблицы, фон waveform, фон списка (WaveformView) |
| 0.05 | Карточки групп в GroupsSheet |
| 0.06 | Master VU контейнер, solo-подсветка рядов |
| 0.08 | Ось waveform, трек VU фон, разделитель рядов, BPM блок |
| 0.10 | ChannelBar (Master VU) фон, выделенная строка, MiniToggle выкл |
| 0.20 | Неактивное время (—:——), подсказка «Пробел» |
| 0.32 | Непроигранная часть waveform |
| 0.35 | GroupColors badge |
| 0.75 | Курсор прогресса waveform |
| 0.85 | Проигранная часть waveform |

### Несоответствия:
1. **VUMeter (трек) vs ChannelBar (мастер)**: трек использует 0.08, мастер — 0.10. Одинаковые элементы должны совпадать.
2. **MiniToggle выкл**: 0.1 — отличается от 0.08 (остальные деактивированные фоны).
3. **Opacity 0.2** для неактивного текста — лучше `.tertiary` для семантики и темы.

### Семантика цветов:
- **accentColor**: основной акцент (BPM, waveform played, progress)
- **orange**: Mute, индикатор «запуск»
- **yellow**: Solo
- **green/red (VU)**: уровень сигнала
- **GroupColors**: blue, green, orange, purple, pink, cyan — предсказуемый порядок ✓
